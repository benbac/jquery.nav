Getting Started

To get started define a basic ul-li list as your navigation however deep you like (note: make sure you define the anchor and span tag within the li tag, it's also recommended to compress the html)

<ul>
	<li>
		<a href="javascript: void(0);"><span>Lorem ipsum</span></a>
		<ul>
			<li><a href="javascript: void(0);"><span>Link 1</span></a></li>
			<li><a href="javascript: void(0);"><span>Link 2</span></a>
				<ul>
					<li><a href="javascript: void(0);"><span>Link 1</span></a></li>
					<li><a href="javascript: void(0);"><span>Link 2</span></a></li>
					<li><a href="javascript: void(0);"><span>Link 3</span></a></li>
				</ul>
			</li>
			<li><a href="javascript: void(0);"><span>Link 3</span></a></li>
			<li><a href="javascript: void(0);"><span>Link 4</span></a></li>
			<li><a href="javascript: void(0);"><span>Link 5</span></a></li>
		</ul>
		<ul>
			...
			...
			...
		</ul>
	</li>
	...
	...
	...
	<li><a href="javascript: void(0);"><span>Link 1</span></a></li>
</ul>

Include the following css file. Adjust styles to your liking.

<link rel="stylesheet" type="text/css" href="css/nav.css">

Include the following js file.

<script type="text/javascript" src="js/jquery.nav.js"></script>

After you include the JS, adjust your CSS and define the HTML call the following in your $(document).ready() block:

$("#nav").nav({columnWidth:140});

Presto!!! That's it, your navigation should work like the above sample!
